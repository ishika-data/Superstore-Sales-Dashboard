# Superstore Sales Dashboard

## Dashboard Preview
![Dashboard](./Superstore%20Sales%20Dashboard.png)

## Project Overview

This project is an interactive Excel Sales Dashboard built using the Superstore Dataset.

### Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

### Dashboard Features
- Total Sales KPI
- Total Orders
- Average Order Value
- Sales by Category
- Sales by Region
- Sales by Segment
- Monthly Sales Trend
- Top 10 Products
- Top 5 Customers

### Key Insights
- Technology generated the highest sales.
- West region recorded the highest revenue.
- Consumer segment contributed the highest sales.
- Canon ImageCLASS Copier was the top-selling product.
